### Webpack-starter

Simple example project for Webpack. Adds the Babel loader and two Webpack plugins.

### Version
1.0.0

### Usage


### Installation

Webpack-Starter requires [Node.js](https://nodejs.org/) v4+ to run.

```sh
$ npm install
```

### Compile
To compile all js to dist/bundle.js

```sh
$ webpack
```
To watch run
```sh
$ webpack -w
```
